Containers

Choose a JDK

  • Container aware

  • OpenJDK 8u192 or above

  • Showed the MaxHeap problem: if you don’t specify, it will take the entire available memory, causing Docker to kill you, to illustrate the non-container aware JDK problem.

    • -XX:+UseParallelGC

  • Two types of memory usage in Java: Heap, Native Memory Usage

  • Still need to configure liveness (are you alive) and readiness (are you ready to accept traffic) probes.
